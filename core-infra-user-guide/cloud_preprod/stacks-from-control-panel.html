You may continue without installing an application at this time and your servers will be created with an empty Django project where you can begin development of your site.


Load Balanced Linux servers running Django with Django-CMS installed into a Python Virtual Environment running behind Apache with mod_wsgi. The backend is a Cloud Database running MySQL.
